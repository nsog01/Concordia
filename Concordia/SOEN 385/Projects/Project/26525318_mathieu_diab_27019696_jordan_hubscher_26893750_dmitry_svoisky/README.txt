---------------
Description
---------------


TelescopeGUI.m and Telescope.fig are the files produced by GUIDE for the GUI running the model.


TelescopeModel.slx is the Simulink model file containing the system.


Scope123.wrl is the VRealmBuilder model file contain the 3D model of the telescope.




----------
Launch
----------


Simply launch the .fig file in Matlab and click on the start button to begin the process.
